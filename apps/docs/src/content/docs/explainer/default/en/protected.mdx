# Protected Page

This page is gated behind authentication. If you can read this, you are signed
in through FerrisKey.

:::callout{variant="info"}
Visitors who are not authenticated are redirected to the login flow, and users
who lack the required role are shown a 404 page in place.
:::

## Restricting by role

To require a specific role instead of any authenticated user, add roles to the
frontmatter:

```yaml
auth:
  enabled: true
  roles:
    - admin
```

A user needs **at least one** of the listed roles to view the page.
