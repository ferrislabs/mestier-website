# Page protégée

Cette page est verrouillée derrière l'authentification. Si vous pouvez la lire,
c'est que vous êtes connecté via FerrisKey.

:::callout{variant="info"}
Les visiteurs non authentifiés sont redirigés vers le flux de connexion, et les
utilisateurs qui n'ont pas le rôle requis se voient afficher un 404 sur place.
:::

## Restreindre par rôle

Pour exiger un rôle précis plutôt que n'importe quel utilisateur authentifié,
ajoutez des rôles au frontmatter :

```yaml
auth:
  enabled: true
  roles:
    - admin
```

Un utilisateur doit posséder **au moins un** des rôles listés pour voir la page.
