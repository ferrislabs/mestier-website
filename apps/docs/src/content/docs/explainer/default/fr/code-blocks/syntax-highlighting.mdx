# Coloration syntaxique

Explainer utilise [Shiki](https://shiki.style/) pour la coloration syntaxique avec prise en charge de deux thèmes — basculant automatiquement entre les thèmes clair et sombre.

## Thèmes

Deux thèmes sont configurés par défaut :

| Mode | Thème |
|------|-------|
| Clair | `github-light` |
| Sombre | `github-dark` |

Le thème actif suit la préférence système de l'utilisateur ou le basculement manuel.

## Utilisation

Utilisez les blocs de code Markdown standard avec un identifiant de langage :

````mdx
```ts
function greet(name: string): string {
  return `Hello, ${name}!`
}
```
````

Rendu :

```ts
function greet(name: string): string {
  return `Hello, ${name}!`
}
```

## Langages pris en charge

Shiki prend en charge [plus de 200 langages](https://shiki.style/languages). Exemples courants :

```bash
echo "Shell scripts"
```

```python
def hello():
    print("Python")
```

```rust
fn main() {
    println!("Rust");
}
```

```json
{
  "key": "JSON"
}
```

## Transformateurs actifs

Explainer active plusieurs transformateurs Shiki pour des fonctionnalités avancées des blocs de code :

| Transformateur | Fonction | Docs |
|-------------|---------|------|
| `transformerNotationDiff` | Marqueurs diff `[!code ++]` / `[!code --]` | [Diffs et focus](/fr/explainer/code-blocks/diffs-and-focus) |
| `transformerNotationHighlight` | Mise en évidence de ligne `[!code highlight]` | [Surlignage de lignes](/fr/explainer/code-blocks/line-highlighting) |
| `transformerNotationWordHighlight` | Mise en évidence de mot `[!code word:name]` | [Surlignage de lignes](/fr/explainer/code-blocks/line-highlighting) |
| `transformerNotationFocus` | Mode focus `[!code focus]` | [Diffs et focus](/fr/explainer/code-blocks/diffs-and-focus) |
| `transformerNotationErrorLevel` | Marqueurs d'erreur `[!code error]` | [Diffs et focus](/fr/explainer/code-blocks/diffs-and-focus) |
| `transformerMetaHighlight` | Surlignage par plage de lignes `{1,3-4}` | [Surlignage de lignes](/fr/explainer/code-blocks/line-highlighting) |
| `transformerMetaLabel` | Labels de blocs de code `[label]` | [Labels et icônes](/fr/explainer/code-blocks/labels-and-icons) |
