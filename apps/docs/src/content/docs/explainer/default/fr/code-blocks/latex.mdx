# Mathématiques LaTeX

Écrivez des équations mathématiques en utilisant la syntaxe LaTeX standard à l'intérieur des blocs de code `math` ou `latex`. Les équations sont rendues au moment du build avec [KaTeX](https://katex.org/) — aucun JavaScript côté client n'est nécessaire.

## Utilisation de base

````mdx
```math
E = mc^2
```
````

Rendu :

```math
E = mc^2
```

## Exemples

### Fractions et intégrales

````mdx
```math
\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}
```
````

```math
\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}
```

### Matrices

````mdx
```math
\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} ax + by \\ cx + dy \end{pmatrix}
```
````

```math
\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} ax + by \\ cx + dy \end{pmatrix}
```

### Sommation

````mdx
```math
\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}
```
````

```math
\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}
```

## Alias de langage

Les identifiants `math` et `latex` sont tous deux pris en charge :

````mdx
```latex
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
```
````

```latex
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
```

## Fonctionnement

Le plugin remark `remarkCodeBlocks` intercepte les blocs de code avec `math` ou `latex` comme langage avant que Shiki ne les traite. La chaîne LaTeX est transmise à un composant Astro `MathBlock` qui appelle `katex.renderToString()` au moment du build, produisant du HTML et du CSS statiques sans aucun JavaScript côté client.

:::callout{variant="info"}
KaTeX prend en charge un large sous-ensemble des commandes mathématiques LaTeX. Consultez les [fonctions supportées par KaTeX](https://katex.org/docs/supported) pour une référence complète.
:::
