# Steps

Les Steps affichent des instructions séquentielles avec une numérotation automatique et un indicateur de progression vertical.

## Utilisation basique

::::::preview
  :::::code-preview
    ::::step-group
      :::step{title="Installer les dépendances"}
        Exécutez la commande d'installation à la racine de votre projet :

        ```bash
        pnpm install
        ```
      :::

      :::step{title="Configurer votre environnement"}
        Copiez le fichier d'environnement d'exemple :

        ```bash
        cp .env .env.local
        ```
      :::

      :::step{title="Commencer le développement"}
        Lancez le serveur de développement :

        ```bash
        pnpm dev
        ```
      :::
    ::::

## Syntaxe

Les Steps utilisent une syntaxe de directives imbriquées :

```mdx
::::step-group
  :::step{title="Titre de l'étape"}
    Contenu de l'étape avec du **Markdown** et des blocs de code.
  :::
::::
```

:::callout{variant="info"}
Les numéros d'étapes sont générés automatiquement. Vous n'avez pas besoin de les spécifier manuellement.
:::

## Props

### step-group

Aucune prop. Sert de conteneur pour les directives `step`.

### step

| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | Titre affiché à côté du numéro de l'étape |
